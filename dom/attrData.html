Attribute Data

Les attributs data, sont des attributs html dynamique. Ce qui est dynamique est leur nom, mais aussi les valeurs qu'on leur donne.
Pour les utiliser, on va ajouter dans la balise html data-*="...".
* = nom de l'attribut, et ... = une valeur
